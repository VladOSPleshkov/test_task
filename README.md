Условные обозначения на игровом поле: "~" - клетка поля, "0" - клетка вышего корабля, "*" - мимо, "X" - попадание. Противник бот бьет по случайной координате, в случае попадания следующим ходом бот попытает добивать цель, по координатам +-xy от предыдущего попадания. 
Совместная игра через общие файлы.
Создайте папку, например: C:\battleship
Откройте к ней общий доступ,
Откройте play_from_shared.bat в блокноте и замените ИМЯ_ПК на имя компьютера игрока 0, и после "\" на имя общей папки:
.jar файл лежит локально на каждом ПК.

Запуск:
игрок 0:
1. Запустите play_from_shared.bat
2. Введите имя
3. Выберите режим: 2 (игра вдвоём)
4. Введите номер игрока: 0
5. Выберите расстановку кораблей
6. Ожидайте подключения противника
Компьютер игрока 1 (после того как игрок 0 дошёл до ожидания):

игрок 1:
1. Запустите play_from_shared.bat
2. Введите имя
3. Выберите режим: 2 (игра вдвоём)
4. Введите номер игрока: 1
5. Выберите расстановку кораблей
6. Игра начнётся автоматически
